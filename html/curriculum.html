HTML & CSS

What is HTML & CSS? These are the two most important and widely used programming languages. Both are HTML and CSS. HTML stands for Hypertext Markup Language and is used to create web pages. CSS stands in for Cascading Style Sheets and allows the web designer to change the appearance of a page. What are these two? What are they used for? How are you supposed to use them? JavaScript

What is javascript? A very basic programming language. It is basically a set of instructions that are sent to the web browser and executed. JavaScript is used for all sorts of things, including web pages and web apps. You can learn more about it in this JavaScript course.



Unity

Unity is a Three Dimensional game engine. Many of the games you play today are built on Unity. Students will learn how to use this game engine and build their own fun and original games, that can be played in real time.

HTML

It is important to learn HTML and Javascript? Yes. HTML is the foundation of the web, so learn it well. You can learn JavaScript in a similar way. If you want to use Javascript, you will need to know how it works in HTML. JavaScript will be a part of HTML, but you should learn Javascript first. The HTML you learn in this course will help you understand JavaScript, and the JavaScript you know will make it easier to create a web application. This course is for beginners. Beginners will learn how to code and create an application with HTML5. In this tutorial you'll learn the basics of JavaScript and how you can create your own applications using JavaScript. You'll start with creating a simple web page. Then you will build on that to develop a full-featured web app.
